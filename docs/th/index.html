ธีร์ธวัช สวาสดิ์ธรรม
Theethawat Savastham

นักศึกษาวิจัย (Research Student) บัณฑิตวิทยาลัยของคณะวิศวกรรมศาสตร์

สวัสดีครับ ผมชื่อ ธีร์ธวัช หรือ ชื่อเล่นคือ ติน, เป็นนักศึกษาวิจัย (Research Student) ที่มหาวิทยาลัยมิยาซากิ ประเทศญี่ปุ่น
ผมกำลังศึกษาเกี่ยวกับ Image Processing และการติดตามวัตถุหลายชิ้น ผ่านกลุ่มของกล้องวีดีโอ ก่อนหน้านี้ผมมีประสบการณ์ในการเป็นนักพัฒนาซอฟต์แวร์แบบ Full Stack โดยเฉพาะ Web Application โดยใช้งาน Node.js Stack อาทิเช่น React, Express, MongoDB ผมมีความสนใจในหลายๆ ด้านทั้ง การพัฒนาโปรแกรม, Cloud Computer, Machine learning, รวมถึงโปรเจคที่เกี่ยวกับความคิดสร้างสรรค์ครับ
