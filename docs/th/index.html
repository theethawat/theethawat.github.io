  • วันเกิด: 7 กุมภาพันธ์ 2541

ตำแหน่งงาน

  • 2565-ปัจจุบัน นักศึกษาระดับปริญญาโท คณะวิศวกรรมศาสตร์ มหาวิทยาลัยสงขลานครินทร์

การศึกษา

  • วิศวกรรมศาสตร์บัณฑิต สาขาวิศวกรรมคอมพิวเตอร์ จากมหาวิทยาลัยสงขลานครินทร์ ปี 2564
  • มัธยมศึกษาปีที่ 3-6 ที่โรงเรียนเบญจมราชูทิศ นครศรีธรรมราช
  • มัธยมศึกษาปีที่ 1-2 ที่โรงเรียนมหาวชิราวุธ จังหวัดสงขลา

ดาวน์โหลดประวัติย่อ ภาคภาษาอังกฤษ

เกี่ยวกับตัวผม

สวัสดีครับ ผมชื่อธีร์ธวัช ตั้งแต่ผมเรียนมัธยม ผมได้ทราบว่าตัวเองรู้สึกหลงไหลในคอมพิวเตอร์ และผมได้มีโอกาสเข้ามาเรียนที่ คณะวิศกรรมศาสตร์ มหาวิทยาลัยสงขลานครินทร์ ใน สาขาวิชาวิศวกรรมคอมพิวเตอร์ ตั้งแต่ปี 2559 ตลอด 4 ปีครึ่งของการศึกษา คอมพิวเตอร์คือหนึ่งในสถานที่ที่ผผมใช้สำหรับการสร้างไอเดียใหม่ ๆ และการจินตนาการต่าง ๆ โดยไม่มีกรอบของมันว่าเป็นไปไม่ได้ จนกระทั่งมันเป็นไปได้

หลังจากจบการศึกษาผมได้ร่วมงานกับ ศูนย์วิจัยระบบอัตโนมัติอัจฉริยะ ซึ่งตั้งอยู่ในคณะที่ผมเรียน ผมทำงานในตำแหน่ง Full Stack Developer ตั้งแต่วันนั้น จนถึงวันนี้ ผมได้สร้าง frontend web application ด้วย React ซึ่งเป็นไลบารี่บนภาษา Javascript ในส่วนของหลังบ้านหรือ Backend ผมได้ใช้ Node.js และ Express.js เป็นเฟรมเวิร์คหลัก สำหรับฐานข้อมูล ผมมักใช้ไปในทาง Non Relational Database เช่น MongoDB มากกว่าพวก relational database. นอกไปจากนี้ ผมต้องทำหน้าที่เป็น Development Operation จำเป็น ผมจะใช้ Docker เป็นหลักในการทำงาน สำหรับการดีพลอยด์แอพพลิเคชั่นของพวกเรา
