ธีร์ธวัช สวาสดิ์ธรรม
Theethawat Savastham

นักศึกษาระดับปริญญาโท สาขาวิชาวิศวกรรมคอมพิวเตอร์

สวัสดีครับ ผมชื่อ ธีร์ธวัช หรือ ชื่อเล่นคือ ติน, เป็นนักศึกษาระดับปริญญาโท ของสาขาวิชาวิศวกรรมคอมพิวเตอร์ตั้งแต่ปี 2565
ผมศึกษาเกี่ยวกับ Neural Network และ AI นอกจากนี้ผมยังเป็นนักพัฒนาซอฟต์แวร์แบบ Full Stack โดยเฉพาะ Web Application โดยผมมักจะสร้างมันด้วย Node.js Stack เช่น
React, Express, MongoDB และสนใจเกี่ยวกับการทำ Microservice ด้วย และผมสนใจการสร้างสรรค์ต่างๆ การพัฒนาโปรแกรม DevOps Cloud Computer และ Machine learning ครับ
